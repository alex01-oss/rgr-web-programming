Завдання:

Створити сайт із застосуванням HTML та CSS. Головна сторінка повинна містити логотип і навігаційне меню, що має 5 пунктів і слугує для переміщення між сторінками.
Сторінок повинно бути 5:

  1. Головна із меню, відомостями про завдання та використаними ресурсами.
  2. Представлення розробника та контактна інформація.
  3. Особисті уподобання, інтереси, їх опис.
  4. Представлення обраної спеціальності або діяльності.
  5. Розповідь про друзів.

В створенні сайту використані, в основному, знання, отримані за семестр. Але для різноманітності були задіяні кілька ресурсів:
- градієнт для тексту: https://html5book.ru/gradient-dlya-teksta
- градієнт для рамки: https://www.internet-technologies.ru/articles/kak-sozdavat-cveta-ramki-s-gradientom-css.html
- позиціонування: https://www.cyberforum.ru/html/thread1479131.html
- відображення: https://developer.mozilla.org/ru/docs/Web/CSS/CSS_Flexible_Box_Layout/Aligning_Items_in_a_Flex_Container
